Transcraft My Accountant
Click here to download the latest version from Sourceforge. The files are:
  • tbook.1.0.7_1.zip for Windows platforms
  • tbook.1.0.7_1.tgz for Linux platforms
  • tbook.jar for just the application. The other prerequisites in order to run up this jar file are followings:
    • log4j 1.2.8 from the Apache Foundation
    • commons-logging from the Apache Foundation
    • jasperreports 1.0.1 from Jasper Reports
    • itext 1.3.1 from Jasper Reports (for PDF generation)
    • db4o 4.6 for Java 1.2 from dbobjects
  • tbook-src.1.0.7_1.tgz for the source code
  • tbook_docs.1.0.7_1.tgz for the documentations
